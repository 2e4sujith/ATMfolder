# ATM Simulation System

## 📌 Project Overview
This is a Java console-based ATM Simulation System developed using Object-Oriented Programming concepts. The application allows users to perform basic banking operations such as creating accounts, depositing money, withdrawing money, transferring funds, and checking account balances.

## 🚀 Features
- Create Account
- View Account Details
- Deposit Money
- Withdraw Money
- Transfer Funds
- Check Balance
- Display Total Accounts
- Custom Exception Handling

## 🛠 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- HashMap Collection
- Exception Handling

## 📂 Project Structure
- ATMOperations (Interface)
- BankAccount (Abstract Class)
- SavingsAccount
- ATMService
- Custom Exceptions

## ▶️ How to Run
1. Clone the repository.
2. Open the project in any Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Compile the Java files.
4. Run the Main class.
